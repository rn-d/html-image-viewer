rnd's html image viewer, ver 0.2alpha, 07/10/2022

INSTRUCTIONS: input folder path where images are (press enter), then in file dialog select multiple images and press enter. Its possible to change display modes (press keys - 1 will touch window from inside, 2 will stretch to window, 3 fit width, 4 original size size) Use left/right keys to cycle images or Home/Delete keys to jump to first/last image.
